# Launch IQ - Admin Guide

## Overview
The Admin Console provides global visibility and control over users, projects, and platform settings. 

## Dashboard & Analytics
- **Activity:** View system-wide activity, recent logins, and project creations.
- **Metrics:** Monitor API usage, token consumption, and system health.
- **Reporting:** Export usage reports for billing and audits.

## User Management
- **Role Assignment:** Assign roles (User, Manager, Admin) to control access.
- **Onboarding/Offboarding:** Suspend or delete users who no longer need access.

## System Configuration
- **AI Models:** Manage API keys for external providers (OpenAI, Anthropic, etc.).
- **Global Settings:** Toggle features like Git integration and export options.
- **Compliance:** Monitor security alerts and platform-wide compliance status.

## Support & Troubleshooting
Check the system logs in the admin dashboard for debugging token failures or API timeouts.
